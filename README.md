# Lasada (ラサダ)

> **「あなたのPCを汚さず、AIにすべてを任せる。安全な『砂場』で動く自律型AIエージェント」**

Lasada（ラサダ）は、AIがあなたの代わりにパソコンの操作やプログラミングを行うツールです。
最大の特徴は、**「Docker（ドッカー）」という隔離された安全な箱（砂場）** の中で作業を行うこと。AIがどれだけ自由に暴れ回っても、あなたの本物のパソコン環境が壊れたり、設定がめちゃくちゃになったりすることはありません。

---

## 🌟 Lasada が選ばれる 3 つの理由

### 1. 壊れない・汚さない「安心感」
すべての作業は「使い捨ての箱（Docker）」の中で行われます。AIが間違って重要なファイルを消すようなプログラムを書いても、被害はその箱の中だけ。本物のPCは無傷です。

### 2. 準備は一瞬、自由は無限
複雑な環境構築は不要です。Dockerさえあれば、AIが自分で必要なツールを「箱」の中に揃え、Pythonで分析したり、Bashでファイルを整理したり、Webで調べ物をしたりします。

### 3. AIに「丸投げ」できる
通常、AIが提案するコマンドは人間が一つずつ確認する必要がありますが、Lasadaなら安全な隔離環境なので「全部自動でやって（--auto-run）」と安心して任せることができます。

## 🚀 クイックスタート

### 1. 準備するもの
- [Docker](https://www.docker.com/) がインストールされていること
- OpenAI などの API キー

### 2. インストール
```bash
git clone https://github.com/kubodad/lasada.git
cd lasada
./install.sh
```

### 3. 動かしてみる
```bash
# 安全な「箱」の中で起動
lasada --sandbox

# AIへの指示例：
# 「今日の天気ニュースを調べて、グラフ付きのレポートを作ってPDFで保存して」
```

## 🛡 安全の仕組み（イメージ図）

AIは「あなたのPC」を直接触るのではなく、専用の「砂場」の中で作業をします。

```text
 [ あなたのパソコン ]
       │
       ▼ （壁があって安全！）
   [ 隔離された砂場 (Docker) ]
       │
       ├─ 🚀 AIが自由にコマンドを実行
       ├─ 🐍 プログラムを書いて動かす
       └─ 📁 ファイルを新しく作る・整理する
```

---

## 💡 こんなことができます

- **プログラミング**: 「Pythonでこんなツールを作って」と頼むと、箱の中で作って動かしてくれます。
- **データ分析**: 「このデータをグラフにして」と頼むと、必要なライブラリを自分で入れて分析します。
- **自動化**: 面倒なファイルの仕分けや、Webからの情報収集もすべてお任せです。
