# Lasada

**Lasada** は、LLM（大規模言語モデル）を用いてタスクを自律的に実行する
ローカルAIエージェント実行基盤です。

プラグインを通じて、AIが直接OSに作用しながら
**計画 → 実行 → 改善** のループを回します。

---

## ⚠️ 注意（必ず読んでください）

Lasadaは以下を実行できます：

* シェルコマンドの実行
* Pythonコードの実行
* ファイル操作
* Webアクセス

これにより、**データの破壊・情報漏洩・意図しない操作**が発生する可能性があります。

👉 **必ずDockerやVMなどの隔離環境で実行してください**
👉 **ホスト環境での直接実行は推奨されません**

---

## 概要

Lasadaは、AIに対して単なる補助ではなく
**実行主体としての役割**を持たせることを目的としています。

従来のツール：

* 人間が操作し、AIは提案する

Lasada：

* AIが判断し、AIが実行する

---

## 主な特徴

* プラグインベースの実行機構（Bash / Python / Web / GUI）
* LLMによる自律的なタスク処理
* ローカル環境での動作（クラウド依存なし）
* Rustによる高速・単体バイナリ構成
* ベクトルDBによる状態保持（RAG）
* `.vlog` による状態駆動型タスク定義

---

## 設計思想

Lasadaは以下の前提に基づいて設計されています：

* AIは誤動作する
* AIの能力は制限すべきではない

そのため、

👉 **AIを制限するのではなく、環境ごと隔離する**

というアプローチを採用しています。
